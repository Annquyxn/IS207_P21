Khởi động QR Server:

  1. Tìm file run_qr_server.bat trong thư mục gốc của dự án (IS207_P21)
  2. Double-click vào file này để chạy
  3. Giữ cửa sổ terminal mở khi sử dụng tính năng thanh toán
  4. Nếu cách trên không hoạt động, mở Windows Command Prompt hoặc PowerShell và chạy các lệnh sau:
cd C:\Users\Admin\Desktop\ok\IS207_P21\python-ChatBot\qrCode
python -m uvicorn main:app --host 0.0.0.0 --port 8000
